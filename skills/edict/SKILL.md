# Edict 下旨

将用户的需求交给太子（`taizi`）agent，启动三省六部流水线。

## 操作步骤

1. 用 Agent 工具启动 `taizi` agent，将用户的需求原文传给它
2. 太子会自动：判断闲聊 vs 旨意 → 提炼标题 → 创建 JJC 任务 → 转交中书省

## 流转路径

```
用户下旨 → 太子分拣 → 中书省规划 → 门下省审议（准奏/封驳）
    → 尚书省派发 → 六部并行执行 → 尚书省汇总 → 回奏
```
